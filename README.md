# CodeAlpha_SalesPrediction

## Project Overview
This project predicts future product sales based on advertising spend across TV, Radio, and Newspaper platforms using Machine Learning regression models.

## Objective
- Predict sales based on advertising budget
- Analyze which ad platform impacts sales the most
- Compare Linear Regression vs Random Forest models
- Deliver actionable insights for business marketing strategies

## Dataset
- Source: Kaggle - Advertising Dataset
- File: advertising.csv
- Records: 200 rows
- Features: TV, Radio, Newspaper ad budgets and Sales (target)

## Tools and Libraries Used
- Python - Programming language
- Pandas - Data loading and cleaning
- NumPy - Numerical operations
- Matplotlib - Data visualization
- Seaborn - Statistical plots
- Scikit-learn - ML models and evaluation

## Models Used
- Linear Regression - Basic prediction model
- Random Forest Regressor - Advanced ensemble model

## Model Performance
- Linear Regression R2 Score: ~0.89
- Random Forest R2 Score: ~0.96
- Best Model: Random Forest

## Visualizations
- Correlation Heatmap
- TV vs Sales, Radio vs Sales, Newspaper vs Sales scatter plots
- Total Ad Spend by Platform bar chart
- Actual vs Predicted Sales plot
- Feature Importance chart

## Key Insights
- TV advertising has the highest impact on sales
- Radio has moderate impact on sales
- Newspaper has the least impact on sales
- Random Forest outperforms Linear Regression

## How to Run
1. Download advertising.csv from Kaggle
2. Open Google Colab
3. Upload the dataset
4. Run sales_prediction.ipynb
